数据库设计文档

数据库名： FIFA

文档版本： V1.0.0

文档描述： 数据库表设计描述

表名 说明
comments 评论表
likes 点赞表
news 新闻表
schedule 赛程表

表名： comments

说明： 评论表

数据列：

序号 名称 数据类型 长度 小数位 允许空值 主键 默认值 说明
1 comment_id int 10 0 N Y 评论ID
2 news_comment_id int 10 0 N N 新闻/评论ID
3 user_id int 10 0 N N 用户ID
4 date timestamp 19 0 N N CURRENT_TIMESTAMP 日期
5 comment text 65535 0 N N 评论

表名： likes

说明： 点赞表

数据列：

序号 名称 数据类型 长度 小数位 允许空值 主键 默认值 说明
1 like_id int 10 0 N Y 点赞ID
2 news_comment_id int 10 0 N N 新闻/评论ID
3 user_id int 10 0 N N 用户ID
4 date timestamp 19 0 N N CURRENT_TIMESTAMP 日期
5 status enum 12 0 Y N 状态（点赞新闻，点赞评论）

表名： news

说明： 新闻表

数据列：

序号 名称 数据类型 长度 小数位 允许空值 主键 默认值 说明
1 news_id int 10 0 N Y 新闻ID
2 title varchar 255 0 N N 标题
3 content text 65535 0 N N 正文
4 date timestamp 19 0 N N CURRENT_TIMESTAMP 日期
5 click_count int 10 0 Y N 点击率
6 like_count int 10 0 Y N 点赞数
7 comment_count int 10 0 Y N 评论数

表名： schedule

说明： 赛程表

数据列：

序号 名称 数据类型 长度 小数位 允许空值 主键 默认值 说明
1 schedule_id int 10 0 N Y 赛程ID
2 home_team varchar 255 0 N N 主队
3 away_team varchar 255 0 N N 客队
4 match_time datetime 19 0 N N 比赛时间
5 match_location varchar 255 0 N N 比赛场地
6 match_status enum 3 0 Y N 比赛状态（已结束，已开售，未开售）
7 tickets_sold_count int 10 0 Y N 售出球票
